A package that will provide to you all the information that you need about the Moon:
  * Moon Azimuth
  * MoonRise Time
  * MoonSet Time
  * Moon Percentage
  * Moon Phase
  * MoonRise Azimuth
  * MoonSet Azimuth
  * Next Full Moon 
  * Next New Moon
