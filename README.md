**MoonKit** is a Swift package which uses math and trigonometry to compute several information about the Moon. 
  
## Features
  * Moon Azimuth
  * MoonRise Time
  * MoonSet Time
  * Moon Percentage
  * Moon Phase
  * Moon Astrological Sign
  * MoonRise Azimuth
  * MoonSet Azimuth
  * Next Full Moon 
  * Next New Moon
